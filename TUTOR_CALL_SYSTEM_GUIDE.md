# Руководство по системе звонков с AI репетитором

## Обзор системы

Система голосовых звонков с AI репетитором использует LMNT API для синтеза речи и обеспечивает:
- Голосовое общение с AI в реальном времени
- Автоматическое списание монет мудрости
- Скидки для премиум подписок
- Историю звонков и статистику

## Экономика монет мудрости

### Стоимость звонков
- **Базовая ставка**: 10 монет мудрости за минуту
- **Минимальная длительность**: 5 минут (50 монет)
- **Максимальная длительность**: 60 минут (600 монет)

## Использование системы

### Для пользователей

1. **Начать звонок**:
   - Перейти на страницу `/tutor-call`
   - Выбрать голос репетитора
   - Нажать "Начать звонок"
   - Разрешить доступ к микрофону

2. **Во время звонка**:
   - Говорите в микрофон
   - AI репетитор отвечает голосом
   - Можно выключить микрофон или звук
   - Таймер показывает длительность и стоимость

3. **Завершить звонок**:
   - Нажать кнопку "Завершить звонок"
   - Монеты будут списаны автоматически
   - Звонок сохранится в истории

4. **Просмотр истории**:
   - Вкладка "История" на странице звонков
   - Статистика: всего звонков, время, потраченные монеты
   - Детали каждого звонка

### Для разработчиков

#### Использование сервиса

```typescript
import { tutorCallService } from '@/services/tutor-call.service';

// Проверить доступность звонка
const affordability = await tutorCallService.canAffordCall(userId);
console.log(affordability);
// {
//   canAfford: true,
//   currentBalance: 500,
//   minCost: 50,
//   discountedCost: 40,
//   discount: 20
// }

// Начать звонок
const session = await tutorCallService.startCall(userId, 'Математика');

// Синтезировать речь
const audioBlob = await tutorCallService.synthesizeSpeech(
  'Здравствуйте! Чем могу помочь?',
  { voice: 'lily', speed: 1.0, stability: 0.5 }
);

// Получить текущий статус звонка
const currentCall = tutorCallService.getCurrentCall();

// Завершить звонок
const finalSession = await tutorCallService.endCall(userId);

// Получить историю звонков
const history = await tutorCallService.getCallHistory(userId, 20);
```

#### API Endpoints

**GET /api/tutor-call?action=check-affordability**
- Проверить возможность совершить звонок
- Возвращает баланс, стоимость и скидку

**GET /api/tutor-call?limit=20**
- Получить историю звонков
- limit - количество записей (по умолчанию 20)

**POST /api/tutor-call**
- Начать новый звонок
- Body: `{ subject?: string }`

**PUT /api/tutor-call**
- Обновить или завершить звонок
- Body: `{ sessionId: string, action?: 'end', durationSeconds?: number, coinsCharged?: number }`

## Автоматическое списание

Система автоматически списывает монеты каждую минуту:
1. При начале звонка проверяется минимальный баланс (50 монет с учетом скидки)
2. Каждую минуту списывается 10 монет (с учетом скидки)
3. Если баланс недостаточен, звонок автоматически завершается
4. При завершении списывается остаток за минимум 5 минут

## Безопасность

### RLS Политики
- Пользователи видят только свои звонки
- Администраторы видят все звонки
- Создание/обновление только своих сессий

### Валидация
- Проверка баланса перед началом звонка
- Проверка владельца сессии при завершении
- Автоматическое завершение при недостатке средств

## Мониторинг и аналитика

### View для статистики

```sql
SELECT * FROM tutor_call_statistics WHERE user_id = 'user-uuid';
```

Возвращает:
- `total_calls` - всего звонков
- `total_duration_seconds` - общее время
- `total_coins_spent` - всего потрачено монет
- `avg_duration_seconds` - средняя длительность
- `avg_coins_per_call` - средняя стоимость звонка
- `last_call_date` - дата последнего звонка

## Troubleshooting

### Проблема: "API key not configured"
**Решение**: Проверьте наличие `VITE_TTS_STT_API_KEY` в `.env`

### Проблема: "Insufficient balance"
**Решение**: Пополните баланс монет мудрости

### Проблема: Микрофон не работает
**Решение**: 
1. Проверьте разрешения браузера
2. Используйте HTTPS (обязательно для микрофона)
3. Проверьте подключение микрофона

### Проблема: Нет звука от AI
**Решение**:
1. Проверьте кнопку динамика (не выключен ли)
2. Проверьте громкость системы
3. Проверьте LMNT API key
