# Исправление белого экрана для администратора на Dashboard

## Проблема
При входе в аккаунт администратора на странице Dashboard отображался белый экран.

## Причина
В файле `src/pages/Dashboard.tsx` была логика, которая перенаправляла администраторов на компонент `SchoolDashboard`. Однако в `SchoolDashboard.tsx` на строке 28 был ранний возврат:

```typescript
if (!currentUser || currentUser.role !== 'parent') return null;
```

Это означало, что для всех пользователей, кроме родителей (включая администраторов), компонент возвращал `null`, что приводило к белому экрану.

## Решение
Переписан файл `src/pages/Dashboard.tsx` с упрощенной логикой:

1. **Удалена зависимость от SchoolDashboard** - администраторы больше не перенаправляются на SchoolDashboard
2. **Создана специальная панель для администратора** - показывает:
   - Приветствие с именем администратора
   - Статистику платформы (название, количество уроков, статус)
   - Быстрые ссылки на:
     - Тикеты поддержки (`/admin`)
     - Все уроки (`/lessons`)

3. **Удален legacy код** - убраны зависимости от:
   - `users` (старый массив пользователей)
   - `resources` (старые ресурсы)
   - `quizResults` (старые результаты викторин)
   - `addChild`, `addResource` (старые функции)

4. **Упрощена структура** - Dashboard теперь работает только с:
   - `profile` - текущий профиль пользователя
   - `lessons` - массив уроков из store
   - `leaderboardService` - для получения рейтинга студентов

## Изменения в коде

### Для администратора
```typescript
if (profile?.role === 'administrator') {
  return (
    <Layout>
      <div className="space-y-8">
        {/* Приветствие */}
        <h1>Привет, {profile.full_name}!</h1>
        <p>Панель администратора</p>
        
        {/* Статистика */}
        <div className="grid grid-cols-3 gap-6">
          <div>Платформа: AILesson</div>
          <div>Уроков создано: {lessons.length}</div>
          <div>Статус: Активна</div>
        </div>
        
        {/* Быстрые ссылки */}
        <Link to="/admin">Тикеты поддержки</Link>
        <Link to="/lessons">Все уроки</Link>
      </div>
    </Layout>
  );
}
```

### Для других ролей
- **Student** - показывает рейтинг, монеты, уроки
- **Teacher** - показывает созданные уроки, монеты
- **Parent** - показывает прогресс детей через `ParentProgressView`

## Результат
✅ Администратор теперь видит свою панель управления вместо белого экрана
✅ Все роли (student, teacher, parent, administrator) корректно отображаются
✅ Удален legacy код, который вызывал ошибки
✅ Нет ошибок TypeScript
