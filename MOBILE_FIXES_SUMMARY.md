### 4. ✅ Создана система кастомных toast-уведомлений
**Файл:** `src/utils/toast.tsx`

Новая система уведомлений заменяет браузерные `alert()`:
- 4 типа: success, error, warning, info
- Красивый дизайн в стиле приложения
- Анимации появления/исчезновения
- Автоматическое закрытие через 5 секунд
- Прогресс-бар показывает оставшееся время
- Кнопка ручного закрытия

**Использование:**
```typescript
import { toast } from '@/utils/toast';

toast.success('Урок успешно создан!');
toast.error('Ошибка при создании урока');
toast.warning('Недостаточно монет');
toast.info('Информационное сообщение');
```

### 6. ✅ Улучшена обработка ошибок AI сервиса

### 7. ✅ Исправлены ошибки API рекомендаций
**Файл:** `src/hooks/useRecommendations.ts`

Проблемы:
- Использовался несуществующий `state.user` вместо `profile`
- Не было обработки ошибок с установкой пустых значений

Исправления:
- Заменено `useStore((state) => state.user)` на `const { profile } = useStore()`
- Добавлена установка пустых массивов/null при ошибках
- Изменено `Promise.all` на `Promise.allSettled` для независимой загрузки данных
