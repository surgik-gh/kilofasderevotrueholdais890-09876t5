# Система звонков с AI репетитором - Резюме

## ✅ Что реализовано

### 1. Экономика монет мудрости
- **Базовая ставка**: 10 монет/минута
- **Минимум**: 5 минут (50 монет)
- **Скидки**: до 30% для премиум подписок
- **Автоматическое списание**: каждую минуту

### 2. Архитектура системы

#### Backend (Сервисы)
- `tutorCallService` - управление звонками
- Интеграция с LMNT API для синтеза речи
- Автоматическое списание монет
- Проверка баланса и доступности

#### Frontend (Компоненты)
- `TutorCallInterface` - интерфейс звонка
- `TutorCallHistory` - история звонков
- `TutorCall` - главная страница

#### API Endpoints
- `GET /api/tutor-call` - история и проверка баланса
- `POST /api/tutor-call` - начать звонок
- `PUT /api/tutor-call` - обновить/завершить звонок

#### База данных
- Таблица `tutor_call_sessions` - сессии звонков
- View `tutor_call_statistics` - статистика
- RLS политики для безопасности

### 3. Функционал

#### Для пользователей

## 📱 Использование

### Начать звонок
1. Открыть `/tutor-call`
2. Выбрать голос репетитора
3. Нажать "Начать звонок"
4. Разрешить микрофон
5. Говорить с AI

### Во время звонка
- Видеть таймер и стоимость
- Управлять микрофоном
- Управлять звуком
- Завершить когда нужно

### После звонка
- Автоматическое списание монет
- Сохранение в истории
- Просмотр статистики
