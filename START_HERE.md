## ⚡ Супер-быстрый старт (для опытных)

### 1. Supabase
```
1. Создать проект на supabase.com
2. SQL Editor → выполнить 3 миграции
3. Скопировать URL + anon key
```

### 2. Groq
```
1. console.groq.com → API Keys
2. Скопировать ключ
```

### 3. Vercel
```
1. Import GitHub repo
2. Framework: Vite
3. Добавить 3 env переменные:
   - VITE_SUPABASE_URL
   - VITE_SUPABASE_ANON_KEY
   - VITE_GROQ_API_KEY
4. Deploy
```

### 4. Обновить Supabase
```
Authentication → URL Configuration
Site URL: ваш-vercel-url
```

### 5. Готово! 🎉

## 🆘 Если что-то пошло не так

### Ошибка сборки
```bash
npm run build  # Проверить локально
```

### AI не работает
Проверить `VITE_GROQ_API_KEY` в Vercel

### Авторизация не работает
Проверить Site URL в Supabase

### Другие проблемы
Смотрите раздел "Устранение проблем" в `DEPLOYMENT_GUIDE.md`

## 🎯 Цели проекта

AILesson Platform - это:
- 🤖 AI-powered платформа для обучения
- 📚 Создание уроков с помощью AI
- 💬 Чат с AI экспертом (Alies)

- 🔔 Система уведомлений
- 👥 Роли: студент, учитель, родитель, администратор

## 📊 Технологии

- **Frontend**: React + TypeScript + Vite
- **Backend**: Vercel Serverless Functions
- **Database**: Supabase (PostgreSQL)
- **AI**: Groq (Llama 3.3 70B)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **State**: Zustand
